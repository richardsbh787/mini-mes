你必须同时有：

① 标准成本（Standard Cost）

来自：

标准工时

标准BOM

标准人工单价

标准设备费率

② 实际成本（Actual Cost）

来自：

实际生产日志

scrap

rework

实际物料消耗

实际工时

二、标准成本计算公式
1️⃣ 标准人工成本
standard_labor =
planned_hours × labor_hour_rate

2️⃣ 标准物料成本
standard_material =
Σ (quantity_required × material_unit_cost)
× planned_output

3️⃣ 标准设备成本
standard_machine =
planned_hours × machine_hour_rate

🔵 标准总成本
standard_total =
standard_labor
+ standard_material
+ standard_machine

三、实际成本计算公式
1️⃣ 实际人工
actual_labor =
actual_hours × labor_hour_rate

2️⃣ 实际物料
actual_material =
Σ (actual_consumption × material_unit_cost)


actual_consumption 包含：

good + scrap + (rework if consume)

3️⃣ 实际设备
actual_machine =
actual_hours × machine_hour_rate

🔴 实际总成本
actual_total =
actual_labor
+ actual_material
+ actual_machine

四、偏差计算（核心）
① 总成本偏差
total_variance =
actual_total - standard_total

② 人工偏差
labor_variance =
actual_labor - standard_labor

③ 物料偏差
material_variance =
actual_material - standard_material

④ 设备偏差
machine_variance =
actual_machine - standard_machine

五、偏差率
variance_rate =
(total_variance / standard_total) × 100%

六、你系统应该新增 API
GET /variance/work-order/{id}


返回：

{
  "standard_total": 10000,

  }
